# WatchGuardian
 This is a simple file tracking application written in go.

## Explanation

This project assigns a change in a specified file as a post request to a specified url.
## Usage

for  run:
```
make run
```
for build:
```
make build
```
for compile:
```
make compile
```

The file path to be followed and the URL to which the change in the file will be sent must be entered while the program is running. Token and header for token can also be entered.
